安装
====================================

    git clone https://github.com/zhouerqin/vimrc.git ~/.vim
    git submodule update --init 
    cp ~/.vim/vimrcs/simple.vim ~/.vimrc 
    cp ~/.vim/vimrcs/editorconfig.vim ~/.editorconfig

为tmux创建一个快照文件
------------------------------------

    :TmuxlineSnapshot ~/.tmux.conf
